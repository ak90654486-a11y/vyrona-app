# Vyrona 🚀

Vyrona is a modern, creator-first global social platform built for connection, creativity, and earning.

---

## 🌍 Vision
To build a secure and scalable social ecosystem for global creators.

---

## ✨ Features
- Home Feed
- Pulse (Short Content)
- Messaging System
- Voice & Video Calls
- Creator Economy
- Wallet System
- Notifications
- Search & Discovery
- Profile System
