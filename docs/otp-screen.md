# Vyrona OTP Verification Screen

## Purpose

Verify user identity securely.

---

## Layout

Top

Vyrona Logo

Title

Verify Your Account

Subtitle

Enter the 6-digit verification code

---

## Components

- 6 OTP Input Boxes
- Auto OTP Detection
- Countdown Timer (60 seconds)
- Resend Code Button
- Verify Button

---

## Security

- Maximum 5 attempts
- Auto expire after 5 minutes
- Firebase OTP Verification
- Device verification

---

## Navigation

Success -> Create Account

Failure -> Retry
